API Endpoints

Method Endpoint Description
POST /api/leads Create a new lead
GET /api/leads List all leads (optional status filter)
GET /api/leads/{id} Get a single lead by ID
PUT /api/leads/{id} Update lead details or status
POST /api/leads/{id}/followup Generate AI follow-up email
POST /api/webhook Receive leads from external systems
